# Rei Barbearia 💈

Sistema de gerenciamento completo para barbearia.

## 📋 Visão Geral

A **Rei Barbearia** é uma plataforma digital que facilita o gerenciamento de agendamentos, clientes, serviços e finanças de uma barbearia moderna.

## 🎯 Funcionalidades

- ✂️ Gerenciamento de serviços
- 📅 Sistema de agendamentos
- 👥 Cadastro de clientes
- 💰 Controle financeiro
- 📊 Relatórios e estatísticas
- 📱 Interface responsiva

## 🚀 Como Começar

### Pré-requisitos
- Node.js (v14+) ou Python (v3.8+)
- Git
- Banco de dados (PostgreSQL ou MongoDB)

### Instalação

1. **Clone o repositório:**
```bash
git clone https://github.com/Nat12459292/Rei-barbearia-.git
cd Rei-barbearia
```

2. **Frontend:**
```bash
cd frontend
npm install
npm start
```

3. **Backend:**
```bash
cd backend
npm install
npm run dev
```
