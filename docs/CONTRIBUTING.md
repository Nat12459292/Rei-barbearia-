# Guia de Contribuição

Obrigado por considerar contribuir para a **Rei Barbearia**! 🙏

## Como Contribuir

### 1. Fork o Repositório
Clique no botão "Fork" no GitHub para criar uma cópia do projeto na sua conta.

### 2. Clone o seu Fork
```bash
git clone https://github.com/seu-usuario/Rei-barbearia-.git
cd Rei-barbearia
```

### 3. Crie uma Branch para sua Feature
```bash
git checkout -b feature/nome-da-feature
```

### 4. Faça suas Mudanças
- Mantenha o código limpo e bem comentado
- Siga as convenções de código do projeto
- Adicione testes para novas funcionalidades

### 5. Commit suas Mudanças
```bash
git add .
git commit -m "feat: descrição da feature"
```

### Convenção de Commits:
- `feat:` - Nova funcionalidade
- `fix:` - Correção de bug
- `docs:` - Documentação
- `style:` - Formatação
- `refactor:` - Refatoração
- `test:` - Testes
- `chore:` - Tarefas gerais

### 6. Push para o GitHub
```bash
git push origin feature/nome-da-feature
```

### 7. Abra um Pull Request
- Descreva as mudanças em detalhes
- Referencie issues relacionadas
- Aguarde a revisão

## Padrões de Código

- Use camelCase para variáveis e funções
- Use PascalCase para classes e componentes
- Adicione comentários em código complexo
- Mantenha funções simples e focadas

## Reportando Bugs

1. Verifique se o bug já não foi reportado
2. Abra uma [Issue](https://github.com/Nat12459292/Rei-barbearia-/issues)
3. Descreva o comportamento esperado vs. atual
4. Adicione prints ou exemplos de código

Obrigado por contribuir! 🎉
